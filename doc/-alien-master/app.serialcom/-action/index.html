Action

data class Action

The Action class is used for exchanging data between the micro-controller and the PC via serial communication.

Action objects will be constructed by parsing the incoming string data using the ActionFromYamlBuilder

values are optional. However, as long as the ActionFromYamlBuilder was used to create the object the values will not be an empty list if the action is of type ActionType.SET or ActionType.REPORT
