MessageParser

class MessageParser

The MessageParser class parses messages coming from the virtual com port from the MBED into one of the two possible messages, Action or StateMessage.

When an object of the two classes is built, the parser will notify every registered listener (registered using addListener) using one of its two methods.

The string passed to the parse method can either be a single line (incomplete message) or a complete message.
