getParameterizedString

fun ResourceBundle.getParameterizedString(key: String, vararg params: Any): String

This method will take the string using ResourceBundle.getString method on the bundle object that is expected to be parameterized with curly brackets, for example "Hello {0} world {1}" The parameters placeholders will be then substituted with actual parameters from the passed collection of params
